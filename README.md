# UKMStartUp

A startup idea-sharing platform built for the Universiti Kebangsaan Malaysia (UKM) community. Students can share startup ideas, receive feedback, and connect with industry partners.

## Features

- **Idea Sharing** — Community members can post and discover startup ideas
- **Pitch Deck** — Idea owners can upload YouTube videos, PDF slides, and cover images to present their startup
- **Custom Sections** — Owners can add custom tabs to their pitch deck (e.g. Q&A, Roadmap, Team)
- **Voting System** — Upvote and downvote ideas
- **Comments** — Community discussion on each idea
- **AI Evaluator** — Groq-powered AI evaluation of startup ideas (score, strengths, improvements, verdict)
- **Industry Interest** — Industry partners can express interest in ideas
- **User Profiles** — Community and industry user types with separate dashboards

## Tech Stack

**Frontend**

- React.js + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

**Backend**

- Node.js + Express.js
- PostgreSQL
- Multer (file uploads)
- JWT Authentication

**AI**

- Groq API (llama-3.3-70b-versatile)
